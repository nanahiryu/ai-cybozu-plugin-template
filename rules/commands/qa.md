# /qa - 全てのテストを実行し品質保証を行う

全てのテストを実行し品質保証を行うコマンド

## タスク

1. テストが全て実行可能か確認する
   - skip されているテストがないか確認する
   - 実装されていない(何も確認していない)テストがないか確認する
   - 一つもテストが実装されていない handler がないか確認する
2. 全てのテストを実行する: `pnpm test`
   - Unit + Integration + E2E を一括実行
3. テスト結果を確認する
4. テスト結果を報告する

## 利用するコマンド

- `pnpm test` - 全てのテストを実行
- `pnpm test:unit` - Unit テストのみ実行
- `pnpm test:integration` - Integration テストのみ実行
- `pnpm test:e2e` - E2E テストのみ実行
