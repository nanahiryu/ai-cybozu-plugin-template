# 技術スタック (Tech Stack)

## コアライブラリ

- **Package Manager**: pnpm
- **Language**: TypeScript
- **UI Framework**: React (v19)
- **Bundler**: Vite

## 品質管理 & 状態管理

- **Linter/Formatter**: Biome
- **状態管理 (State Management)**:
  - React Hooks (基本)
  - Jotai (複雑な状態管理が必要な場合)

## テスト

- **Unit Test**: Vitest
- **E2E Test**: Playwright
