# プラグイン設定の反映手順

## 問題

プラグイン設定画面で設定を保存しても、レコード詳細画面に反映されない。

## 原因

プラグイン設定を保存しただけでは、kintoneアプリに反映されない。
「アプリを更新」が必要。

## 解決手順

1. プラグイン設定画面で設定を保存
2. 「アプリの設定」画面に遷移
3. 「反映前の変更があります」のメッセージを確認
4. **「アプリを更新」ボタンをクリック**
5. 確認ダイアログで「アプリを更新」をクリック
6. 更新完了後、レコード詳細画面で動作確認

## E2Eテストでの対応

`helpers/pluginConfig.ts` の `savePluginConfigAndDeploy()` を使用すると、
設定保存 -> デプロイ -> デプロイ完了待機を一括で行える。

```typescript
await savePluginConfigAndDeploy(page, APP_ID, PLUGIN_ID, config);
```
