# Role

E2Eテストの実装・実行を担当

# Task

1. テスト仕様書（`docs/spec/test-spec.md`）を確認
2. 対象ハンドラに対応するE2Eテストを実装
3. Playwright MCP を利用して動作確認しながら実装
4. E2Eテストを実行し、結果を報告
